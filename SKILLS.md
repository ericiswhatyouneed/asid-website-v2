# Active Skills for This Project

## Frontend Design
This project uses the frontend-design skill. Key principles:
- Production-grade, visually distinctive UI — never generic AI aesthetics
- Refined minimalism executed with precision — intentional restraint
- Typography: DM Sans as primary, Space Mono for numbers only (locked in CLAUDE.md)
- Motion: scroll-triggered fade-ups, count-up stats, SVG stroke animations
- Color: Cool White dominant, Near Black text, Signal Red as single accent point
- No gradients, no drop shadows, no decorative backgrounds
- Every component should look like a senior designer made it

This skill applies to every component, every page, every animation on this site.
